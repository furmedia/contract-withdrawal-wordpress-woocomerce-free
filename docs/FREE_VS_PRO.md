# Free versus Pro

The two editions are separate plugins and separate repositories. The free repository must never contain Premium implementation hidden behind a licence key or payment gate.

| Capability | Free | Pro add-on |
|---|---:|---:|
| Guest and signed-in declaration form | Yes | Uses/extends Free |
| Full and partial order declarations | Yes | Uses/extends Free |
| Immediate customer and merchant email | Yes | Yes |
| Protected text evidence | Yes | Yes |
| Basic merchant list/detail | Yes | Yes |
| Customer My Account history | No | Planned |
| Operator statuses, comments and status email | No | Planned |
| Automatic email retries and cron/API runner | No | Planned |
| Multiple administrator recipients | No | Planned |
| Filtered CSV exports | No | Planned |
| RTF and print/PDF model forms | No | Planned |
| Delivery diagnostics | No | Planned |
| AI-assisted operations | No | Planned service/add-on |

The Free edition must remain useful on its own. Pro features are implemented only in the commercial repository.

# Privacy and data

The plugin stores the submitted identity, email address, contract reference, withdrawal scope, selected items, optional note, generated declaration, submission timestamp, configuration snapshot and email evidence. It does not store raw visitor IP addresses; a keyed one-way bucket is used only for rate limiting.

No telemetry, external updater, licence server or AI service is used by this free edition. Email delivery uses the site’s WordPress mail configuration.

Merchants must document the processing in their privacy information, choose a retention policy, restrict administrator access and handle data-subject requests according to their own legal obligations. Uninstall intentionally preserves evidence and settings.
